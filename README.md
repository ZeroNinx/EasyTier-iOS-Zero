# EasyTier iOS 15+ Jailbreak Fork

[简体中文](README_CN.md) | [English](README.md)

This is a personal iOS 15+ jailbreak-oriented fork of the original EasyTier iOS branch.

### Overview

This fork is used to explore an iOS 15-compatible client path for jailbroken devices. System-level permissions and App Store-oriented capabilities from the original branch may be removed or replaced with jailbreak-specific implementations over time.

### Installation

No public release channel is provided by this fork. Build and install it manually in a local development or jailbreak environment.

### Status

This repository is a personal fork and is not an official EasyTier release.
